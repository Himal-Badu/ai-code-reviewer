# AI Code Reviewer
AI-powered code review tool for detecting bugs, security issues, and code quality improvements.

## Installation
```bash
pip install ai-code-reviewer
```

## Usage
```bash
ai-reviewer review /path/to/code
ai-reviewer scan --security /path/to/code
```
